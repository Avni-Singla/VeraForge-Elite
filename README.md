# 🚀 VeraForge Elite

AI-powered Merchant Engagement Engine built using FastAPI.

VeraForge Elite helps businesses reactivate dormant customers, trigger intelligent campaigns, and automate engagement workflows.

## 🌐 Live Demo

https://veraforge-elite.onrender.com

## 📘 API Docs

## ✨ Features

- Smart merchant context ingestion
- Trigger-based campaign recommendations
- Automated customer reply handling
- Health monitoring endpoints
- Metadata service
- Interactive homepage
- Public cloud deployment
